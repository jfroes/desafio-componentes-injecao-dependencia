# Desafio: Componentes e Injeção de Dependência

💻 Projeto do curso Java Spring Professional, do professor Nélio Alves, desenvolvido como parte de um desafio para demonstrar o uso de componentes e injeção de dependência no Spring Boot.

---

## 🔍 Visão Geral

Este projeto mostra como criar e integrar componentes no Spring Boot, utilizando o mecanismo de **Injeção de Dependência** para promover baixo acoplamento e modularidade.

## 🚀 Tecnologias

- Java 21
- Spring Boot
- Maven


## 📁 Estrutura do Projeto

- `src/main/java/` – implementação dos componentes, serviços e configuração do Spring Boot
    - Exemplos de anotações: `@Component`, `@Service`, `@Autowired`
- `pom.xml` – gerenciamento de dependências e plugins de build (Maven)
